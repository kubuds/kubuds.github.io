AOSP

AOSP Solutions

Assist enterprises to quickly complete AOSP system development adaptation.

Provides CVE patch maintenance and related services to ensure the security of upstream and downstream AOSP systems.

Service List

AOSP system migration and adaptation

Based on hardware ecosystem devices, transplant and develop the corresponding Android basic system, maintain the corresponding version of AOSP source code, and ensure that the function, stability, performance and security are up to commercial standards.

Continuous maintenance of CVE

In accordance with the Android Security Bulletin(ASB), we will actively maintain patches for the corresponding versions of CVE, including platform fixes, Linux kernel fixes, and patches provided by SOC manufacturers such as AMLogic, MediaTek, Qualcomm.
At the same time, we provide backporting downupstream versions to ensure the security of upstream and downstream AOSP systems.
