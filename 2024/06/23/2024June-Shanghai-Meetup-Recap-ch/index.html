活动回顾｜6月上海欢聚会，聊了哪些厉害的东西？

2024-06-23 苦芽科技

6 月上海线下欢聚会昨天顺利举行，本次聚会由苦芽科技承办组织，超过 30 多位网友从多个城市赶来，分享并交流了编译器、机器人、RISC-V 锦标赛、甲辰计划文档工程等多个热门话题。

openEuler ROS SIG Maintainer、中国科学院软件研究所王晓云工程师发表演讲《openEuler ROS 发展&应用&展望》

王晓云工程师首先概述了开源机器人操作系统 ROS 的当前状况，其开发者活跃度日均达十万，为行业创造了数十亿美元的价值。她随后回顾了 ROS SIG 的起源和 openEuler ROS 的发展历程，并重点介绍了最新进展：openEuler 24.03LTS 版本已发布超过 1000 个 ROS 软件包，实现了对 ROS1/ROS2 双版本及 X86/ARM/RISC-V 多架构的支持。

王晓云工程师通过视频向与会者展示了 openEuler ROS 的显著成果，包括无人车和其他机器人的应用案例，如 RISC-V MicroROS 小车和生肖机器人等。她对 openEuler ROS 的未来发展充满期待，希望其能够得到持续的维护和功能扩展，适配更多开发板，并得到社区的积极推动，实现更多实际应用。作为 openEuler ROS SIG 的首席导师，王晓云还分享了 ROS 机器人开发实习生计划，并鼓励有兴趣的学生积极参与。（点击左下角 阅读原文 了解更多详细信息）

openEuler RISC-V SIG Maintainer、苦芽科技 CTO 王俊强分享《LLVM 平行宇宙：迈向多星文明，公布跃迁路径》的演讲

王俊强先生首先介绍 LLVM 平行宇宙计划的发展历程，并简要介绍了依托于 openEuler RISC-V SIG 和 openEuler Compiler SIG 在过去一年中的进展。

紧接着，王俊强介绍了本月新启动的 openKylin LLVM 平行宇宙计划，并展开介绍了依托于「甲辰计划开源实习生联合招聘培养项目」建立的 J126 实习生岗位内容。王俊强号召更多的高校学生加入到 LLVM 平行宇宙这个很有意思的项目中来。

王俊强还向观众介绍了 KarsierOS，KarsierOS 是苦芽科技基于 openEuler LLVM 平行宇宙的成果衍生出的下游商业发行版，预计在 8 月 21 日 RISC-V 中国峰会上正式发布。这可能是第一个基于平行宇宙成果的商业发行版。

最后，也是本次发布最为吸引人的地方，王俊强先生首次公开了「跃迁路径」工程计划：用大约五年的时间，实现「全体系软件供应链」的编译器、链接器、基础库按需切换的自由。

中科院软件所 OERV 团队周嘉诚工程师分享《openEuler LLVM 平行宇宙计划介绍与进展》的演讲。

周嘉诚首先回顾了 LLVM 平行宇宙计划项目的背景，即开源社区以 LLVM Clang 构建 Linux 发行版的历程，随后对平行宇宙项目做了基本介绍：此项目旨在尝试用 LLVM 编译器套件替代 GCC 套件，构建 Linux 发行版与软件包，独立于社区版本并行发展。而这也是 openEuler 社区将此项目命名为「平行宇宙」的原因。

这之后他分享了 LLVM 的几大优势和选择其的原因，包括其优秀的架构设计、友好的协议以及活跃的生态系统。此外他表示 LLVM 在代码质量检测、对 Rust 内核驱动开发的原生支持，以及在工业界和学术界的广泛认知等方面也很有竞争力。

其后周嘉诚工程师详细介绍了 openEuler 平行宇宙计划目前状况和运作方式，包含构建过程中切换 LLVM/Clang 编译器的方式、当前切换编译器涉及的软件包范围基础设施情况等。

最后他分享了 openEuler 平行宇宙计划最近发布新版本的情况、下载链接以及后续大致计划。

第二届 RISC-V 软件移植及优化锦标赛（RVSPOC）组委会主席王文君女士在本次活动上宣布报名暨比赛开始。

王文君女士首先肯定了第一届 RISC-V 软件移植及优化锦标赛取得的成就，最终收获了 20 多份高质量的 PR，评出了 10 组冠军。

随后王文君女士宣布第二届 RISC-V 软件移植及优化锦标赛报名暨比赛开始。后续由苦芽科技赛博酸橙团队负责人，同时也是第一届锦标赛主席李程先生做赛题的预发布。

苦芽科技赛博酸橙团队负责人、首届锦标赛主席李程先生做《第二届 RISC-V 软件移植及优化锦标赛赛题预发布》

李程先生发布了第二届 RISC-V 软件移植及优化锦标赛赛题，其中包括“新 32 位 Type-1 Hypervisor 移植”、“Llama 3 在 TH1520 上的优化实现”，“OceanBase 移植与优化”，“TDengine 移植与优化”，“RetroArch 移植与优化”。李程先生对每个赛题都做了简单的介绍，提出了每个赛题对应的要求以及评分准则，并且强调当前赛题处于预发布阶段，后续可能会动态调整。

除此之外，李程先生在会议休息期间公布了“甲辰计划开放 J128 Deepin 操作系统开发实习生（RISC-V 架构）｜开源实习生联合招聘培养”的消息，并作为该实习岗位的首席导师号召感兴趣的同学加入实习。

苦芽科技社区运营顾问章翔分享《从实习到创新：我的技术建设与社区之路》的演讲

章翔先是简单介绍了一下自己与 ROS、以及 RISC-V 的一些渊源，从 2018 年读研的时候实习开始接触了 ROS，在 2021 年加入 PLCT 实验室接触了 RISC-V，直到今年选择加入苦芽科技，开辟新的职业方向，从事社区运营。

他进一步分享了对社区建设的一些观点，同时他表示要在社区运营的岗位做有意义的事，致力于推动 RISC-V ROS 社区的建设，营造良好的社区交流环境，努力打造培养技术人才的土壤环境。

章翔重点介绍了 riscvpedia 的相关情况，这是一个定位为 RISC-V 使用者的百科全书，收录所有市面上的 RISC-V 产品和相关软件信息的开源 git 仓库，计划每 3 个月更新一次。他简单介绍了 riscvpedia 的框架与组织架构，并表示目前均处于初期规划阶段。最后，章翔展示了 riscvpedia 的目标：在 2036 年到来之际，希望在超过 5000 万的 RISC-V 使用者中，有超过 100 万的 riscvpedia 注册成员，并呼吁更多人参与。

最后，章翔代表苦芽科技宣布加入 ROS SIG，将在甲辰计划「联合实习生招聘培养工程」框架下，定向开放 50 名开源实习生名额参与欧拉社区 ROS SIG 的建设，竭力为 ROS 生态的发展做出更多的贡献。
