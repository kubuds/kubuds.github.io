CEO 李威威在 RISC-V 生态实验室及开源生态建设研讨会上发表演讲

2025年01月07日 苦芽科技

2025 年 1 月 6 日，南京英麒智能科技有限公司主办，上海苦芽科技有限公司协办的「RISC-V 生态实验室及开源生态建设研讨会」在南京英麒智能科技有限公司春分会议室成功举办。此次研讨会汇聚了多位行业精英，共同探讨 RISC-V 生态的建设与发展。苦芽科技 CEO 李威威受邀参会，并在会上发表了题为“QEMU for RISC-V 现状、进展、在 RISC-V Lab 中的结合讨论”的精彩演讲。

在此次研讨会上，李威威首先对南京英麒智能科技有限公司的热情邀请表示了诚挚的感谢。他表示，这次研讨会为 RISC-V 生态的建设与发展提供了一个良好的交流平台，使各方可以分享和探讨在 RISC-V 生态建设中的经验和见解。同时，他强调了南京英麒智能科技有限公司在 RISC-V 生态建设中的重要地位和贡献，尤其是在南京地区 RISC-V 生态实验室及甲辰计划南京漂流基地等项目的建设上所取得的显著进展。

李威威先生（右一）正在作报告（左一为甲辰计划主理人吴伟先生）

随后，李威威详细介绍了 QEMU 在 RISC-V 架构中的应用现状。他表示，目前 QEMU 对 RISC-V 的支持已经相对完善，能够模拟多种 RISC-V 架构的处理器，并支持多种操作系统和应用程序的运行。同时，他也指出了 QEMU 在性能优化、功能扩展等方面仍存在一些挑战和改进空间。

在谈到 QEMU 的进展时，李威威分享了苦芽科技在这一领域的研究成果和实践经验。他提到，苦芽科技通过不断优化和改进，提升了 QEMU 的性能和稳定性。此外，苦芽科技还积极参与开源社区的建设，与其他开发者共同协作，推动 QEMU 的功能扩展和创新。

李威威还就 QEMU for RISC-V 在 RISC-V Lab 中的结合进行了深入讨论。他认为，RISC-V Lab 作为一个开放的实验平台，为 QEMU 的应用和推广提供了良好的环境。同时，QEMU 也为 RISC-V Lab 提供了强大的模拟和仿真能力，使得实验室能够更好地支持 RISC-V 生态的创新和发展。

此次研讨会的成功举办，进一步加深了业内对 RISC-V 生态建设的凝聚力。苦芽科技作为 RISC-V 生态的重要参与者，将与各方携手共进，共同推动 RISC-V 生态的蓬勃发展，助力中国在全球信息产业领域实现更大突破。同时，作为「甲辰计划」中的一员，苦芽科技呼吁更多企业和开发者加入这一致力于 RISC-V 生态发展的计划，共同推动 RISC-V 生态的持续发展与壮大。

苦芽科技衷心感谢南京英麒智能科技有限公司的邀请，期待未来在 RISC-V 生态建设中与英麒智能及其他合作伙伴开展更深入的合作与交流。
