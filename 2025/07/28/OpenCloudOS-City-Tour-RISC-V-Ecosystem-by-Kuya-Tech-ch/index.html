工程师孙敏参加OpenCloudOS 城市行活动，助力RISC-V生态发展

2025年07月28日 苦芽科技

2025 年 7 月 26 日，OpenCloudOS 城市行活动首次在成都举办，苦芽科技工程师孙敏受邀出席，并积极互动，与各行业专家分享 RISC-V 发展状况。本次活动议程主要围绕 x86 架构国产化适配：包括芯片算力、芯片算力如何接入操作系统、数据库国产化，现有架构的国产化适配方向及难点，对新兴 RISC-V 架构有诸多借鉴价值。

活动热闹非凡-议题满满干货

孙敏在会后第一时间分享参会感受：

  • 硬件级别的加密计算，性能与安全性不可兼得，为了让操作系统的加密计算接口保持最新，有必要构建一个专门软件源。
  • OpenCloudOS 的云原生技术深挖业务痛点，解决了容器版本的碎片化问题，大大减少容器的创建、升级、备份、迁移成本。
  • 以 OpenCloudOS 为例，优秀的虚拟化技术需要打通硬件虚拟能力、操作系统透传、虚拟化软件三重壁垒。
  • 国产数据库在实现基本的增、删、改、查功能后，需要紧跟国内不同行业数据库业务的痛点，往行业聚焦，并且尽可能做到开箱即用。

苦芽科技将持续跟踪 OpenCloudOS 社区动态，积极参与 RISC-V 生态建设，为国产芯片、操作系统、数据库的深度融合贡献更多“苦芽方案”。

关于工程师孙敏

在嵌入式开发行业拥有 10 余年从业经验，从事过安卓应用开发，端测机器学习；2022 年进入 RISC-V 开发社区，从事过 Debian Java Team 修包、openEuler RISC-V 打包，AOSP RISC-V RVV 优化；参与主办过历届 RVSPOC 锦标赛；目前就职于上海苦芽科技有限公司，任职研发工程师，担任 openGauss RISC-V sig 维护者，OpenCloudOS RISC-V sig 组成员，甲辰计划开源实习生 mentor，业余爱好是羽毛球运动。
