苦芽科技亮相2025 RISC-V Tokyo Day

2025-02-28 苦芽科技

2 月 27 日，2025 RISC-V Tokyo Day 在日本东京成功举办。作为 RISC-V 开源生态的新成员，【苦芽科技 KUBUDS】以顶级赞助商身份亮相。甲辰计划主理人兼苦芽联合创始人吴伟发表主题演讲，全面展示苦芽科技的成长历程。同时现场展台展示了多项前沿技术成果。

主题演讲

了解全部演讲内容，请点击阅读：

《演讲公开｜甲辰计划主理人在 RISC-V Day Tokyo 2025 介绍甲辰计划的历史与路线图》

甲辰计划：机制、进展与愿景

吴伟作为甲辰计划主理人，详解运作细节，以及对未来的计划：

  • 运行机制：无董事会、无会员费，完全依赖社区自治。任何企业、开发者均可自由加入或退出，通过 GitHub 协作推动项目。
  • 核心进展：已有 40+ 家厂商与 500+ 名开发者加入，落地“开发板漂流计划”、“开源实习生联合培养”等项目，累计移植优化超 200 款软件至 RISC-V 平台。
  • 目标愿景：到 2036 年，推动 1000 款关键商业软件完成 RISC-V 适配，覆盖 AI、边缘计算、医疗等核心领域，终结“先发者劣势”。

苦芽科技诞生：源于一群年轻人的 RISC-V 梦想

苦芽科技作为甲辰计划的积极参与者，一直致力于编译器、模拟器乃至操作系统等基础软件生态建设，以“为 RISC-V 提供商业级技术支撑”为使命，迅速成为众多高性能 RISC-V 芯片和厂商的生态软件合作伙伴。主理人介绍：“苦芽的成长证明，开源精神与商业价值可以共存。我们既是技术贡献者，也是生态受益者。”

展台亮点

本次苦芽科技的展台上，展出了多项创新项目与技术方案。甲辰计划首次在海外全景展示技术版图。

黑科技演示

现场演示 5 分钟内将 x86 应用迁移至 RISC-V 平台。

集成 LLVM 编译器、Gem5 仿真器及 V8 运行，支持端侧 AI 模型一键部署。

全栈软件服务矩阵

  • 模拟器定制开发：基于 Spike、QEMU、GEM5 等开源工具，提供从指令集仿真到全系统模拟的验证服务，加速 RISC-V 芯片硬件开发，支持 SystemC TLM2 标准模型。
  • RISC-V 系统适配：覆盖 OpenHarmony、openEuler、RT-Thread 等多种操作系统，实现从 RTOS 到 Linux 的深度优化，助力客户快速部署 各种定制版及工业级嵌入式方案。
  • 工具链优化：基于 GCC/LLVM 的跨架构（x86_64/ARM/RISC-V）适配服务，集成自动向量化、AI 加速库（MLIR/TVM），性能显著提升。
  • 嵌入式芯片一站式方案：为 ARM/RISC-V 提供裸机系统移植、驱动开发、专属加速库优化及平台性能评估，覆盖机器人、工业场景等多个领域。

甲辰计划部分成果展示

  • 商业软件移植：联合 100+芯片厂商与 500+ISV，已完成 200+关键软件 RISC-V 适配，涵盖 AI、边缘计算、工业仿真等领域。
  • 开发者生态创新：展示“开发板漂流计划”东京节点成果，累计发放超 80 块设备，玄铁、Milk-V 等厂商捐赠积分破 26 万点。
  • 人才储备行动：开源实习生项目已培养 350 名高校生，参与 40+项目，包括 Linux 内核、V8 引擎等核心组件开发。

致谢与展望

苦芽科技向 RISC-V 大使傅炜、东京组织方、PLCT Lab、Milk-V、Sipeed表示诚挚感谢。

参与本届 Tokyo Day，是苦芽科技在 RISC-V 全球化布局中迈出的关键一步。随着甲辰计划的持续推进，中国开源力量正为 RISC-V 生态注入更多可能性。
