共建系列 | openKylin走进上海苦芽共同探讨RISC-V生态

2025-12-12 openKylin

为进一步增进社区合作伙伴间的友好交流及合作探讨，OpenAtom openKylin 社区共建系列之走进上海苦芽共同探讨 RISC-V 生态交流活动成功召开。来自达摩院玄铁、超睿科技、知合计算、进迭、SIFive、赛昉等 10 余家企业参会代表共聚苦芽，介绍了 RVA23 的进展、当前 RISC-V 国际、国内的生态进展和最新成果，并基于 RISC-V +openKylin+苦芽的生态共建等话题展开了深入交流和沟通。

嘉宾致辞

上海苦芽 CEO 李威威表示：非常感谢 openKylin 社区提供给上海苦芽承办这次 openKylinRISC-V+苦芽生态共建活动，也感谢与会嘉宾来参加这次活动，苦芽未来会持续维护并更新 ROS SIG，也会根据社区需要创建并加入新的 SIG，持续为 RISC-V 生态添砖加瓦。最后预祝本次活动圆满成功！也祝 openKylinRISC-V 生态越来越繁荣。

分享环节

openKylin 社区生态合作负责人马发俊向与会伙伴分享了社区的最新进展、运作模式与关键突破。目前，openKylin 正携手 2000 多家社区共建伙伴，共同打造下一代操作系统技术创新与繁荣生态。展望未来，在各方伙伴的聚力共建下，openKylin 社区将加速成长，矢志跻身国际顶尖开源社区行列，协力锻造全球领先的智能桌面操作系统开源根社区。

openKylin 社区硬件生态合作负责人朱晓红向与会伙伴分享了 openKylin 在 RISC-V 系统版本、及 RVA23 规划、生态建设与 AI 融合等方面取得的进展与成果，并展望面向 RISC-V 未来的发展规划。

RISC-V 基金会推广大使、上海苦芽联合创始人吴伟在《RISC-V 全球生态介绍》主题分享中重点回顾了过去一年全球 RISC-V 软件生态的里程碑进展，并特别展示了国内团队在 RISC-V+AI、机器人、汽车电子等领域的最新突破性成果；也介绍了过去 1 年来全球 RISC-V 社区的进展和突破，并重点介绍了甲辰计划各成员单位在过去一年中取得的成就，包括 openKylin 在 RISC-V 国际基金会知名度稳步提升等喜人成绩。

openKylin ROS SIG Owner、上海苦芽科技工程师李程在《苦芽助力 ROS 与 openKylin 生态共建》主题分享中回顾了加入社区后，在社区成立 ROS SIG ,积极参与社区贡献并于 2025 年 10 月取得重大突破—— 成功在 openKylin 上运行 ROS 经典应用“小乌龟” 。未来，ROS SIG 将重点完成更多软件包适配、维护上游更新，并基于 openKylin 及开发板打造演示 Demo，持续推动社区 ROS 生态发展。

达摩院玄铁 RISC-V 高级技术专家王云龙在《玄铁在 RVA23 上的软件建设进展》主题分享中系统回顾玄铁与 openKylin 在 RISC-V 生态建设中的深度合作成果，重点展示双方在高性能桌面操作系统优化、主流应用软件移植适配及 AI 软件栈协同开发方面的突破性进展。同时，将首次披露玄铁在 RISC-V RVA23 软件栈的前沿建设：涵盖架构标准制定、固件产品化开发实践、在国内开源操作系统社区的核心贡献，以及针对 RVV 向量扩展的深度软件优化技术。通过软硬协同创新，共同推动 RISC-V 在通用高性能领域的产业化落地，为构建 RISC-V 高性能软件生态提供关键技术支撑。

超睿科技高级软件工程师胡远韬在《RISC-V + openKylin 跑通 VR 游戏》主题分享中重点介绍了基于 RISC-V (DP1000) + openKylin 的全栈开源 VR 解决方案。通过整合 Godot 开源引擎、WiVRn 开源串流工具以及 rvsaber 节奏游戏，openKylin + UltraRISC 成功打通了从底层硬件到上层应用的完整环节，验证了 RISC-V 生态在 VR 游戏领域的运行能力。

知合计算产品经理程宇在《知合计算与 openKylin 生态合作分享》中重点介绍了知合计算作为国内 RISC-V 的领军企业，在端侧布局了 a210 等大模型算力芯片。同时和 openKylin 深度合作，在云终端，具身智能等产品拓展应用场景。

RISC-V 生态共建讨论环节

在 RISC-V 生态共建讨论环节，与会嘉宾对共建生态都表现出了极高热情，积极发言，献言献策。从技术创新、技术贡献、生态贡献、市场推广等多维度基于 openKylin 的 RISC-V 的未来，伙伴间更深层次合作方向，分享了各自的见解和建议。与会嘉宾达成高度共识，为 RISC-V 生态共建携手前行，共同为 RISC-V 好用的生态添砖加瓦。
