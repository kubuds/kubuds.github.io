苦芽科技出席2025 OpenCloudOS生态大会，分享RISC-V生态建设成果

2025-12-07 苦芽科技

2025 年 12 月 6 日，在 OpenCloudOS 操作系统生态大会上，上海苦芽科技有限公司联合创始人吴伟、研发工程师孙敏受邀登台，以 “三年蓄势，直奔 RVA23” 为核心，分别就甲辰计划布局、OpenCloudOS RISC-V SIG 成果展开深度分享。

吴伟：甲辰计划推动 RISC-V 生态人才与产业协同

作为甲辰计划主理人、RISC-V 大使吴伟首先解读甲辰计划核心目标：在 2036 年（丙辰年）前，完成所有基础关键行业领域的 RISC-V 适配与优化，联合超 100 家芯片厂商、500 家软件企业，构建超万人规模的顶尖 RISC-V 人才网络。

目前该计划已联合 PLCT 实验室、Milk-V、算能等伙伴推动全产业链覆盖，截至 2025 年 9 月成员单位超 50 家，还创新推出 “开发板随缘漂流计划”—— 通过 “厂商 / 个人捐赠 - 开发者申领 - 实践应用” 的流程，降低开发者入门门槛，助力 RISC-V 技术普及。吴伟同时提到，苦芽科技作为 RISC-V 开源软件生态 “填坑者”，将持续依托甲辰计划链接产业资源。

孙敏：OpenCloudOS RISC-V SIG 三年成果显著

随后，openGauss riscv 维护者、aosp-riscv 贡献者孙敏，详细展示了 OpenCloudOS RISC-V SIG 的成立背景与技术支持成果。SIG 成立源于双重契机：一是 RISC-V 已成长为继 x86、Arm 后的第三大 CPU 架构，生态逐步成型；二是信创背景下，北京市科委、信创工委会对 OpenCloudOS 社区及 RISC-V 技术的支持。而 OpenCloudOS 与 RISC-V 在 “开源驱动、开放治理、国产替代、简洁可裁剪” 等维度高度互补，为合作奠定基础。

据孙敏介绍，经过三年深耕，SIG 已达成多项里程碑：适配 VisionFive 1、Megrez、URDP1000 等 5 款 RISC-V 开发板，发布 4 个通用 OS 镜像；完成 2500+baseos 软件包、5400+epol 软件包构建，累计解决 151 个技术 Issue；桌面与 Mesa 领域适配完成度达 45%，为图形界面应用提供支撑。技术层面，SIG 选定 Linux 6.6 长期支持内核，既满足信创 “CPU 可控、无出口管制风险” 要求，又强化 RISC-V 在服务器 / 桌面场景的支持，实现基础显卡（Gnome 桌面）、Chromium 浏览器、ffmpeg 视频编解码、KVM 硬件虚拟化等功能落地，在超睿 DP1000 硬件上可高效运行混元大模型（Hunyuan-0.5B-Instruct-Q4_K_M）。

展望未来：从开源社区到产业落地

孙敏透露，SIG 将以 RVA23 标准为核心推进适配，重点布局 AI、机器学习、HPC 领域，计划 2026 年 Q2 发布 RVA23 预览镜像；同时联合腾讯系应用（如腾讯会议、微信后台）开展商业落地，并通过 “Fork/PR 参与开发、微信群交流、甲辰实习生计划、厂商加入甲辰计划” 等多元渠道，邀请全球开发者共建自主可控、安全高效的 RISC-V 软硬件生态。

大会现场同步分享了 RISC-V 入门资源，推荐了中国科学院软件研究所 PLCT 实验室打造的编译原理、操作系统、模拟器开发等系列课程，以及 B 站 “从零开发 RISC-V 操作系统”“移植 V8 引擎” 等实操教程，为人才培养提供支撑。详细内容在以下链接
