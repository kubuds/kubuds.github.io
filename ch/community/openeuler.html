聚焦 openEuler「LLVM平行宇宙」计划

上海苦芽科技有限公司于2024年3月1日签署了贡献者许可协议(Contributor License Agreement,CLA),正式加入openEuler社区。

之前,公司联合创始人、CTO王俊强长期担任 openEuler RISC-V SIG maintainer,在 21.09 至23.09 等版本的 RISC-V 适配中承担了社区领导者的角色，并与 openEuler Compiler SIG 共同发起了「LLVM平行宇宙」计划 (基于LLVM技术栈构建openEuler版本) 。

之后,苦芽计划加强在 openEuler RISC-V SIG 和 Compiler SIG 的贡献投入,聚焦于「LLVM平行宇宙」计划的加速完成并正式发版。
