CEO 李威威受邀出席 2024 CCF 中国软件大会，探讨 RISC-V 技术发展

2024-11-15 苦芽科技

西安，2024 年 11 月 15 日 —— 苦芽科技首席执行官（CEO）李威威受邀出席 2024 年中国计算机学会（CCF）主办的中国软件大会，并在“RISC-V 系统软件及软硬协同前沿展望论坛”中发表演讲。该论坛汇聚了 RISC-V 产业界的商业、教育及技术领袖，旨在深入探讨 RISC-V 系统软件的发展趋势、科研挑战、技术难点、商业机遇等关键议题，以促进 RISC-V 技术在国内的进一步发展。

在此次论坛中，李威威先生发表了题为“RISC-V 模拟器前沿进展：以 QEMU 为例”的报告。他首先介绍主流 RISC-V 开源模拟器的总体现状，特别是 QEMU 在 RISC-V 架构支持上的发展历程，包括国内所做的主要贡献。

李威威先生还详细介绍 QEMU 在 RISC-V 架构上的支持特性及基本原理，并结合业界内 QEMU 的具体应用场景，展示了 QEMU 在 RISC-V 领域的前沿进展。

最后结合社区 RISC-V QEMU Sig 中的规划，对 QEMU 在 RISC-V 方面的未来发展进行展望。

作为论坛的特邀嘉宾，苦芽科技 CEO 李威威表示：“我们非常荣幸能够参与此次中国软件大会 RISC-V 前沿论坛。RISC-V 作为开源技术的重要组成部分，正在全球范围内推动技术创新和产业变革。苦芽科技期待在 CCF 中国软件大会上与业界同仁共同探讨 RISC-V 技术的未来，为推动国内 RISC-V 生态的建设贡献力量。”

苦芽科技是一家专注于 RISC-V 架构研发和应用的高新技术企业。公司致力于推动 RISC-V 技术在各个领域的应用，通过技术创新和生态合作，为全球客户提供高性能、高可靠性的 RISC-V 解决方案。

本次出席中国软件大会，不仅代表了苦芽科技在 RISC-V 领域的深厚技术积累和行业影响力，也预示着苦芽科技将在推动 RISC-V 技术发展和生态建设中发挥更加积极的作用。作为 RISC-V 生态中的活跃力量，苦芽科技一直致力于通过技术创新，推动 RISC-V 架构在更广泛领域的应用，特别是在高性能计算和人工智能领域。
