Android RISC-V 两年内迎来爆发期，苦芽科技为合作伙伴提供全方位支持

2024年06月17日 苦芽科技

在 6 月 13 日世界嵌入式系统大会的 RISC-V 分论坛上，甲辰计划主理人吴伟先生首次透露了甲辰计划对于未来 10 年的 RISC-V 市场渗透率预测。其中移动计算部分的占有率将会从 0% 增涨到 40% 以上。这意味着 Android (AOSP) RISC-V 将在两年内迎来第一次产品爆发期。

自 2022 年起，谷歌公司与 RISC-V 国际基金会已经建立了良好的合作基础，并与高通等公司 2023 年联合成立了 RISE 基金会用于加速包括 AOSP 在内的开源软件生态对 RISC-V 的支持。但是 AOSP 庞大的软件规模，以及 RISC-V 目前在移动市场的后来者为止，使得谷歌一家对于 RISC-V 的投入支持，可能无法跟得上 RISC-V 创新创业者们日新月异的迭代速度。

在几个月前，谷歌公司的某个内核团队就因为产品迭代压力尝试过暂时移除 RISC-V 的支持，又在三周后撤销了移除补丁，重新加回了 RISC-V 架构支持。谷歌发言人及时进行了声明，表示 Android 将继续支持 RISC-V，明确表示了对 RISC-V 的长期承诺并未改变。这一声明缓解了市场对于 Android 放弃 RISC-V 的担忧，但也透露出在 Android 全面支持 RISC-V 之前，确实还有大量的研发及工程工作需要进行。即使谷歌准备好在内核中重新添加 RISC-V 支持，也需要时间和精力来确保与各种 RISC-V 设备的完美融合。这意味着，如果只是等待 AOSP upstream 的更新的话，我们可能需要等待比原先预期更长的时间（3 年以上），才能看到搭载 RISC-V 芯片的商业 Android 设备问世。而甲辰计划将有希望将预期时间从原先的“3 年以上”缩短到“1 年以内”。

苦芽科技将密切关注 Android 对 RISC-V 支持的进展，并与 RISC-V 厂商紧密合作，帮助伙伴加速产品研发迭代速度，缩短产品推向市场的周期。苦芽科技的工程师团队在 Linux 内核、操作系统发行版、AOSP 系统框架、模拟器及调试器工具领域都有着丰富的经验，并且在过去五年间参与过 AOSP RISC-V 开源移植工程的多项支撑工作。同时，作为甲辰计划的成员单位，苦芽科技也将继续推动 RISC-V 生态的发展，助力 RISC-V 在商业领域的广泛应用。
